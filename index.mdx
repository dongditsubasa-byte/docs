SunnyMeteo（小晴天科技）为开发者与企业提供**高精度气象、气候与环境数据**服务，支持 REST API、SDK、MCP 与地图能力。

## 基础信息

| 项目 | 说明 |
|------|------|
| 生产 API 基址 | `https://api.sunnymeteo.com` |
| 协议 | HTTPS + JSON（UTF-8） |
| 鉴权 | `X-API-Key` 请求头 |
| 稳定前缀 | `/v1/...` |
